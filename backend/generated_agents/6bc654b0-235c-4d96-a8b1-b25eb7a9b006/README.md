# Loan Underwriting Pipeline

This project implements a loan underwriting pipeline using the CrewAI framework. It processes loan documents, verifies identity, checks income and employment, analyzes credit, and aggregates risk factors to generate a final risk score and recommendation.

## Prerequisites
- Python 3.11+
- Required API keys or configurations should be set in `config.yaml` or as environment variables.

## Installation
```bash
pip install -r requirements.txt
```

## Configuration
- `config.yaml`: Configure each agent and tool. Ensure all necessary API keys and configurations are provided.

## Run Command
```bash
python main.py
```
